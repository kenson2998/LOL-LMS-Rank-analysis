## 前言 ##
學習Django的時候，看到英雄聯盟有各國伺服器的數據戰績網，但是台灣的數據就是沒有人做出來，<br/>
於是抱著學習的心態去製作它，邊做邊學中發現很多東西會運用和經驗就將它記錄下來。

<ul>
	<li>語言:Python3.6</li>

	<li>資料庫:mongodb</li>

</ul>
